HowTo - Install NuGet package in Visual Studio

This tutorial explains how to manually install the ChillSharp NuGet package in Visual Studio by downloading it from GitHub and binding a local NuGet package source.

Overview of the Process

What you are about to do

  • Download the ChillSharp NuGet package from GitHub;
  • Place it in a local folder on your machine;
  • Register that folder as a NuGet source in Visual Studio;
  • Install the package like a normal NuGet dependency.

Once configured, Visual Studio will treat your local folder exactly like an official NuGet feed.

Step 1 - Download ChillSharp from GitHub

Go to the ChillSharp GitHub repository and download the NuGet package manually.

Step 2 - Create a Local NuGet Folder

  • Create a dedicated folder on your machine for local NuGet packages:
    For example: C:\NuGet\LocalPackages
  • Then copy the downloaded ChillSharp.x.y.z.nupkg file into this folder.

You can store multiple packages here if you want. Visual Studio does not care. It will index everything.

Step 3 - Register the Local NuGet Source in Visual Studio

Now you need to tell Visual Studio that this folder exists and should be treated as a NuGet feed.

Open NuGet Package Source Settings:

  • Open Visual Studio
  • Go to Tools ‚Üí Options
  • Navigate to NuGet Package Manager ‚Üí Package Sources

Add a New Source:

  • Click the + button
  • Set: Name: "Local NuGet", Source: "C:\NuGet\LocalPackages"
  • Click OK

Congratulations. You have successfully taught Visual Studio where files live on your own computer.

Step 4 - Install the NuGet Package

Now comes the familiar part.

Open the NuGet Package Manager

  • Right-click your project
  • Select Manage NuGet Packages

Select the Correct Package Source looking at the top-right of the NuGet window:

  • Open the Package source dropdown
  • Select Local NuGet (or whatever name you gave it)
  • Search for ChillSharp
  • Select the package
  • Accept the license agreement if prompted

Visual Studio will now reference the package exactly as if it were installed from nuget.org.
